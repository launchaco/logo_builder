Launchaco AI Logo Builder Demo

This simple interface allows the changing of a feature vector to see what fonts are most similar to that vector. All fonts used are licensed under 'Open Font License' and can be seen in the fonts directory. A full dataset of all fonts used on www.launchaco.com/logo can be seen in the dataset directory :) We use a k-d tree for storing our 7 dimensional dataset.
